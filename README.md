# ancient-book-server
基于Springboot+Vue前后端分离的古籍数字化网站，系统包含管理员、用户角色，功能如下： 管理员：首页统计（古籍统计、七天活动报名统计）；用户管理；古籍分类；古籍管理；板块管理；帖子管理；帖子评论；资讯管理；活动管理；活动报名；留言反馈；系统公告；网站指南；个人信息；密码修改。 用户：注册、登录系统；首页；古籍展览（详情、收藏）；论坛交流（板块、帖子、评论、点赞）；古籍资讯（详情）；线下活动（详情、报名）；系统留言；系统公告；网站指南。用户还具有个人中心功能，功能包括：我的古籍、古籍收藏、我的帖子、帖子评论、帖子点赞、活动报名、我的留言、个人中心、修改密码。 系统界面美观大方，功能丰富，使用了SpringBoot、Vue、ElementUI、Echarts等技术栈，有需要可提供指导。
